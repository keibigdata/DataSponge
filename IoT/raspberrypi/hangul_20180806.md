### 한국어 적용  
  
- 그래픽 화면에서 시작(라즈베리모양 클릭) > ? > Raspberry Pi Configuration으로 가서
- Raspberry Pi Configuration이 실행되면 Localisation 탭을 클릭하고  
- Set Locale...을 클릭하고 Language는 ko (Korean)으로 Character Set은 UTF-8으로 변경한다.  
(재시작은 바로 해도 되고 아래 내용 진행 후 해도 된다.)  
- 소프트웨어 저장소 정보를 업데이트 하고  
- 한글 폰트와 관련한 기본 폰트를 설치한다. : sudo apt-get install ibus ibus-hangul fonts-unfonts-core
